# resume-skill

**从任意 HTML 模板风格中，快速生成专业、美观、A4 单页可打印的简历。**

> 别再从零开始写简历了。你已经有一个喜欢的 HTML 设计风格，我们直接把它"翻译"成高质量的简历模板。

---

## 解决什么问题？

- 看到好看的 HTML 模板，想做成简历但不知怎么下手
- 做出来的简历打印出来溢出两页，或底部太空
- PDF 导出后颜色全丢了
- 想要中英文两个版本，但风格要完全一致
- 每次做简历都要重新调间距、头像大小、颜色

这个 skill 已经把这些坑都踩过一次，打包成了可复用的模板 + 流程。

---

## 快速开始

### 方式一：直接用模板（推荐）

1. 复制 `templates/` 下的模板文件：
   - `resume-a4-zh.html` — 中文模板
   - `resume-a4-en.html` — 英文模板
2. 把所有 `[占位符]` 替换成你的真实信息
3. 替换头像（建议 90×90px）
4. 浏览器打开，按 `P` 键打印预览，微调即可

### 方式二：从参考风格生成

当你有一个具体的 HTML 模板（幻灯片、作品集、设计系统等），按以下流程操作：

1. **分析风格** — 提取颜色、字体、间距、设计调性
2. **基于模板生成** — 以 `templates/` 中的模板为骨架，套用提取到的设计系统
3. **迭代打磨** — 重点调整底部留白、头像大小、PDF 颜色保真

## 模板说明

| 文件名              | 说明                 | 用途             |
|---------------------|----------------------|------------------|
| `resume-a4-zh.html` | 中文 A4 单页模板     | 国内投递         |
| `resume-a4-en.html` | 英文 A4 单页模板     | 海外投递         |

**模板特点**：
- 严格控制在一页 A4
- 内置 PDF 颜色保护（`print-color-adjust: exact`）
- 头像默认 90px，可直接修改
- 所有需要替换的内容用 `[占位符]` 标注
- 结构：关于我 → 能力 → 经历 → 项目 → 亮点 → 教育

---

## 使用建议

**替换占位符的推荐顺序**：
1. 姓名 + 职位 + 联系方式
2. 关于我（建议先写 2-3 版再挑最好的）
3. 工作经历（每条 bullet 尽量带数据或结果）
4. 项目与成果 + 其他亮点
5. 技能标签（建议不超过 8-9 个）
6. 教育和认证

**打印测试 Checklist**：
- [ ] 只占 1 页
- [ ] 底部没有明显大块留白
- [ ] 强调线和技能标签颜色正常显示
- [ ] 头像清晰不模糊

## 常见问题

**Q: PDF 颜色丢了怎么办？**
模板已内置 `print-color-adjust: exact`，浏览器打印时颜色应正常保留。如果仍丢失，请确认使用 Chrome / Edge 打印。

**Q: 内容放不下一页怎么办？**
优先缩减"关于我"字数 → 减少 bullet 条数 → 缩小 section 间距。头像可从 90px 缩至 72px。

**Q: 怎么改成其他颜色？**
修改 CSS 中 `:root` 的 `--accent` 变量即可全局换色。
